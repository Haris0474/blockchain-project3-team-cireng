# Simple Voting Smart Contract

## Deskripsi
Project smart contract voting sederhana menggunakan Solidity dan Hardhat.

## Fitur
- Owner dapat menambahkan kandidat
- User dapat vote satu kali
- Event logging
- Candidate tracking

## Cara Menjalankan

### Install
npm install

### Compile
npx hardhat compile

### Testing
npx hardhat test

### Run Local Blockchain
npx hardhat node

### Deploy
npx hardhat run scripts/deploy.js --network localhost

## Teknologi
- Solidity
- Hardhat
- Ethers.js
- MetaMask
